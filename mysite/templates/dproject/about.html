What is this?

This website helps users find and create transcripts to YouTube videos.

How to use

Copy and search by YouTube link or install browser bookmark.

Purpose

  • Access for the deaf and hard of hearing. Greater than 5% of the total world population has significant hearing loss (source).
  • Make video more discoverable. Search engines (like Google) work best with text.
  • Open a video to a larger audience by translating a video into another language.
  • Browse without sound.

Purpose

Features are always being added or refined.
